# rn-assignment7-11117540

The Open Fashion app is a React Native application designed to provide a seamless shopping experience. Users can browse products, view product details, add items to their cart, and proceed to checkout. The app features a customizable drawer navigation for easy access to different sections.

Features
Home Screen: Displays featured products and categories.
Cart Screen: Shows selected items, allowing users to modify their cart and proceed to checkout.
Product Detail Screen: Provides detailed information about a selected product.
Drawer Navigation: Allows easy access to various sections such as Home, Cart, Product Detail, Store, Locations, Blog, Jewelry, Electronics, and Clothing.

Installation
Clone the Repository:

git clone https://github.com/yourusername/open-fashion-app.git
cd open-fashion-app


Install Dependencies:
npm install


Install React Native CLI (if not already installed):
npm install -g react-native-cli

Install Expo CLI (if not already installed):
npm install -g expo-cli

Run the Application:
npx react-native run-android  # For Android
npx react-native run-ios      # For iOS

Usage
Start the Application:
npx react-native start

Open the Application on your emulator or connected device.


Custom Fonts: The app uses the TenorSans font loaded asynchronously.
Navigation: The app uses react-navigation for managing different screens and navigation drawers.
Image Assets: Ensure all images and fonts are correctly placed in the respective directories (./assets).
